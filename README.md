# ❤️ Heart Disease Risk Factor Analysis using Tableau & Flask

## 📌 Project Overview

Heart disease is one of the leading causes of mortality worldwide. This project analyzes key risk factors contributing to heart disease using interactive data visualizations created in Tableau and integrated into a Flask web application.

The goal of this project is to transform raw health data into meaningful insights that help healthcare professionals, policymakers, and individuals make informed preventive decisions.

---

## 🎯 Objectives

- Analyze heart disease prevalence across different demographics.
- Identify key lifestyle risk factors such as smoking and physical inactivity.
- Examine medical conditions like diabetes and stroke in relation to heart disease.
- Develop interactive dashboards and data stories using Tableau.
- Integrate Tableau visualizations into a Flask web application.

---

## 🛠️ Tools & Technologies Used

- **Tableau Public** – Data visualization and storytelling
- **MySQL Workbench** – Database storage and SQL operations
- **Flask (Python)** – Web integration
- **VS Code** – Development environment
- **HTML & CSS** – Frontend structure
